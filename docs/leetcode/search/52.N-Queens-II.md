## 52. N-Queens II

- [Link to LeetCode](https://leetcode.com/problems/n-queens-ii/)

**Description:**



The **n-queens** puzzle is the problem of placing `n` queens on an `n x n` chessboard such that no two queens attack each other.

Given an integer `n`, return *the number of distinct solutions to the **n-queens puzzle***.

**Constraints:**

- `1 <= n <= 9`



<!-- tabs:start -->

#### **Solution 1**



```java
class Solution {
    private int[] cols, diag1, diag2;
    int ans;
    public int totalNQueens(int n) {
        cols = new int[n];
        diag1 = new int[2 * n - 1];
        diag2 = new int[2 * n - 1];
        
        ans = 0;
        dfs(n, 0);
        return ans;
    }
    private void dfs(int n, int y) {
        if (y == n) {
            ans += 1;
            return;
        }
     
        for (int x = 0; x < n; ++x) {
            if (cols[x] != 0 || diag1[x + y] != 0 || diag2[x - y + n - 1] != 0) continue;
            cols[x] = 1;
            diag1[x + y] = 1;
            diag2[x - y + n - 1] = 1;
            dfs(n, y + 1);
            cols[x] = 0;
            diag1[x + y] = 0;
            diag2[x - y + n - 1] = 0;
        }
    }
}
```



> // DFS W/ Backtracking + Pruning
>
> Time complexity: O(N!)
>
> 放置第 1 个皇后有 N 种可能的方法，放置两个皇后的方法不超过 N (N - 2) ，放置 3 个皇后的方法不超过 N(N - 2)(N - 4) ，以此类推。总体上，时间复杂度为 : O(N!).
>
> Space complexity: O(N)
>
> 需要保存对角线和行的信息。
